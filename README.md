# CareQ - Hospital Management System

CareQ is a modernized hospital management dashboard system featuring a premium, dark-themed, glassmorphism-inspired interface. It consists of a Node.js/Express backend and a React/Vite frontend.

## Features
- 🏥 **Comprehensive Dashboards**: Tailored views for Staff, Patients, and Admins.
- 🎨 **Premium Aesthetic**: Dark mode, glassmorphism UI, smooth animations, and responsive design.
- 🔒 **Secure Operations**: Advanced session management, rate-limiting, and XSS protection.
- ⚡ **Real-Time Updates**: Instant data visualization and status tracking.

## Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (v16.0 or higher)
- [Git](https://git-scm.com/)

### Installation

1. **Clone the repository:**
   ```bash
   git clone <your-github-repo-url>
   cd cie
   ```

2. **Setup the Backend:**
   ```bash
   cd backend
   npm install
   # Start the server (runs on http://localhost:5000)
   npm start
   ```

3. **Setup the Frontend:**
   ```bash
   cd ../frontend
   npm install
   # Start the UI (runs on http://localhost:5173)
   npm run dev
   ```

## Tech Stack
- **Frontend**: React, Vite, Tailwind CSS (or Vanilla CSS with modern styling), Framer Motion (if applicable).
- **Backend**: Node.js, Express, SQLite (via `careq.db`).
